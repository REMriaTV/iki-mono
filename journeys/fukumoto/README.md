# Fukumoto Lab: 探究と教育の旅
**テーマ：水は誰のもの？見えない水をさがせ！**

## 概要
「水」と「鴨川」をテーマとしたラーニング・ジャーニー。
参加者が生物多様性の重要性を五感で体感し、その保全への行動へとつながるプログラムを設計・試行します。

### 2025-12-18: 京料理 二傳リサーチ
  <a href="/iki-mono/journeys/research_niden.html">research_niden.html</a>
ゆき＆りえ、お店訪問してヒアリング
#### 視点
- 魚でも海魚と川魚に別れる。佃煮を売ってる店は川魚が多い。
- 比叡山・鴨川の風土とお茶の文化の間で、日本料理はどう発展してきたか。
- 京料理と生物多様性。鴨川との結びつき、お茶との結びつき
- 鴨川との距離感。実際の当日ルート。
- 四季折々を庭や料理に取り入れる。
- 山の幸を運ぶ流域（水運）は鴨川。出汁を引く軟水も鴨川。
- 仕出分野と茶懐石の文化
#### ヒアリング 
- 水が溢れてた。小川通りに昔、川が流れていた
- 堀川は昔、川だった。堀川三条に、材木を置くための材木屋があり、細長い造りになった
- 西高瀬川（嵐山）。と別橋で端に細い川がある。千本三条で堀川に移す
- 海魚（かいぎょ）は持ってくる間に腐ってしまう
- 「いかりけ」関東では熟成魚ともいう。締めてから旨みが出る。身のぷりぷり感は失われる
- 「かいこう」みが柔らかくなるうちにアミノ酸が出てくる。関東ではこの時が食べ頃
- 関西では「いかりけ」のある、新鮮でイキの良い魚が好まれる。新鮮な魚が貴重だった名残か
- たけうちせいほう。小池も水が流れてた。小川の小池あたりに実家があった。水浸し。実家が仕出し屋
- 伊藤若冲も錦市場の八百屋の息子さん
- 御池通りが拡張したのは戦争の時で、元は今の半分くらいほどしかなかった。建物はすべて疎開させられた
- 掘れば井戸水が出てくるし、川から水を引くのも簡単だった。淡水魚は水があれば生きていける
- 「はも」いかりけが長く続く生命力の強い魚。まな板の上でも噛みついてくるほど
- 産卵直前に餌を食べ込んで肥えて旬となる。産卵期を終えると身が枯れて味が変わる。
- 鱧は8月に産卵を終えた9月にまた太る。旬は６〜7月。9月のを「新鱧」と呼ぶ。
- 痕跡を辿っていくと楽しい。まさに探偵！伝承の背景に面白さがある
- 探偵ポイントはどこか。「身の回りにある"見えない水"を探してきて」
- 天保にほって、慶応に印刷した地図。おどいが残っている。江戸をほぼそのまま表している
- 鴨川が西に蛇行して、京極のあたりを通っていた
- 三条の〜〜にその痕跡がある。川は上から下へ流れる
- 運動会で綱引きをすると、南が勝つ。南北に傾斜した京都の地形を表す逸話。
- 街の中の気づかない傾斜がある。見えない川の道。
- 鱧と相性の良い食べ物
- 代表的なのは松茸。梅肉。青梅。
- 相性が良くても、季節が離れてたら出会えない。タイミングは重要。
- 12月はおせち作りをしている。
