# Nakahigashi Lab: 食と職人の旅
**テーマ：水が育む京都の食文化を味わう旅をつくろう！**

## 概要
「水」と「食」のつながりを可視化する旅。
料理人、工芸職人、食品加工業者と共に、自身の事業に「見えない価値（水・自然）」がどう関わっているかを探求します。

> 「実は地下水でやってます。地下水が出るから京都文化が育まれた。

> 繊細な和食の文化を育んだのは水であることは間違いない。」
