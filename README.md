# 生きものむすぶ・みんなのミュージアム：体験共創アーカイブ

## プロジェクト趣旨
「京都の文化と自然の生態系をどう紐解くか」をキーワードに、食の専門家である中東篤史さんと学びの専門家である福本理恵さんを招いて、「生きものむすぶ・みんなのミュージアム」プロジェクトを実施します。
本サイトは、その試行錯誤と「プロセスそのものを可視化するアーカイブ・サイト」です。

世界屈指の文化都市・京都を舞台に、「文化と自然の生態系」に生きる豊かさを体感する体験共創プラットホームの創出を目指します。

---

## Learning Journeys (試行実施)

### [🧪 Fukumoto Lab: 探究と教育の旅](./journeys/fukumoto/README.md)
**テーマ：水は誰のもの？見えない水をさがせ！**
京都を舞台に探究学習の第一人者とともに環境教育体験の作り方を自ら経験しながら学ぶプログラム。

### [🍽️ Nakahigashi Lab: 食と職人の旅](./journeys/nakahigashi/README.md)
**テーマ：水が育む京都の食文化を味わう旅**
京都の豊かな水とそれによって育まれてきた食文化、そしてそれを支える人々の技術を深く探求するプログラム。
